🔧 Technical Details

URL Parameter Format: index.html?album=ALBUM_NAME&song=SONG_FILENAME

  • album: Must match an existing album directory name exactly
  • song: Must match an existing audio file name in the specified album (including extension)
  • Error Messages: Clear, user-friendly error messages for invalid parameters
  • Validation: Both parameters are validated against available albums and songs
